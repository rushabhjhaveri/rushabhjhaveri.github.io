Pocket Pharmacist

HackRU Spring 2017
  • This project is an Android app that aims to spread awareness about drug-to-drug interaction.
  • The app takes text input or even a photograph of a printed prescription as input.
    The app then returns the drugs (both medicinal and recreational) that interfere with the drugs/medication you're taking, along with the level of severity of the interactions.
  • The app makes use of the REST API and RXNorm, fetches data from the NIH database, and uses Microsoft's ComputerVision API.
